What is HTML?

HTML is the standard markup language for creating Web pages.
HTML stands for Hyper Text Markup Language and describes the structure of a Web page.
HTML consists of a series of elements which tell the browser how to display the content and are represented by tags.

The tags label pieces of content such as "heading", "paragraph", "table", and so on Browsers do not display the HTML tags, but use them to render the content of the page
